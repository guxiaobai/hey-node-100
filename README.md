# Hey Node 100



## macOS

> `Sequoia`

```bash
brew install node@22
```

> Apple silicon
```bash
fish_add_path /opt/homebrew/opt/node@22/bin
```

> `Intel-based`
```
fish_add_path /usr/local/opt/node@22/bin
```

choco install nodejs-lts --version="22.12.0"
